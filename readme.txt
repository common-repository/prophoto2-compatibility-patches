ProPhoto2 compatibility patches for WordPress 3.0+. Only for users of the ProPhoto2 theme.

== Description ==

Only for users of version 2 of the ProPhoto theme. Contains patches required for compatibility with WordPress 3.0+

== Installation ==

1. Upload `p2-wp-compatibility` folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
